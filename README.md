## API Documentation

#### Returns all items

```http
  GET /api/items
```

| Parameter   | Type       | Description                           |
| :---------- | :--------- | :---------------------------------- |
| `api_key` | `string` | **Mandatory**. Your API key |

#### Returns an item

```http
  GET /api/items/${id}
```

| Parameter   | Type       | Description                                   |
| :---------- | :--------- | :------------------------------------------ |
| `id`      | `string` | **Mandatory**. The ID of the item you want |
